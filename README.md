# Student Grade Manager (C)

## Description
A console-based C program that manages student grades for multiple modules.  
It allows users to input or load data from a file and calculates the final weighted average based on exam and TD scores.

The project uses linked lists to store module information dynamically.

---

## Features
- Input student modules manually or from a file
- Store module data (name, coefficient, exam mark, TD mark)
- Calculate weighted average based on customizable weights
- Save data to a file (optional)
- Dynamic memory management using linked lists

---

## How It Works
Each module contains:
- Subject name
- Coefficient
- Exam mark
- TD mark

Final grade is computed using:
- Exam percentage (user-defined)
- TD percentage (user-defined)
